# Simple Todo App - プロダクト仕様書

## プロダクト概要

**目的**: シンプルで直感的なタスク管理アプリ  
**ターゲット**: 個人利用者（学生・社会人・フリーランス）  
**価値提案**: 必要十分な機能を美しいUIで提供し、タスク管理を楽しくストレスフリーにする

---

## ユーザーストーリー

### メインユーザー: 忙しい社会人（田中さん、28歳）

- 状況: 仕事とプライベートのタスクが混在して頭の中がごちゃごちゃ
- 課題: 既存のタスク管理アプリは機能が多すぎて使いこなせない
- ニーズ: シンプルかつ美しいUIでタスクを分類管理し、進捗を可視化したい

### 使用シーン

1. 朝の通勤時: 今日やることをカテゴリ別に整理して追加
2. 仕事中: 思いついたタスクを詳細付きで記録
3. 夜寝る前: 今日完了したタスクをチェックして達成感を味わい、統計を確認

## UI/UX 要件

### デザイン原則

1. 美しさ: モダンで洗練されたビジュアル
2. 直感性: 説明不要で操作できるインターフェース
3. 気持ちよさ: 滑らかなアニメーションとマイクロインタラクション
4. 包容性: ライト・ダークモード対応で時間帯を選ばない

### ビジュアル仕様

| 要素           | ライトモード                                           | ダークモード                                               | 理由                         |
| -------------- | ------------------------------------------------------ | ---------------------------------------------------------- | ---------------------------- |
| 背景           | グレー系グラデーション (gray-50 → stone-50 → slate-50) | スレート系グラデーション (slate-900 → gray-900 → zinc-900) | 集中力を妨げない美しい背景   |
| カード         | 半透明白 (white/90) + 微妙な影                         | 半透明スレート (slate-800/50)                              | モダンなガラスモーフィズム   |
| テキスト       | ストーン系 (stone-800/600/500/400)                     | スレート系 (slate-100/300/400/500)                         | 可読性と階層の明確化         |
| アクセント     | エメラルド (emerald-600)                               | エメラルド (emerald-400)                                   | 完了時の達成感を演出         |
| アイコン       | Lucide React                                           | Lucide React                                               | 一貫性のあるアイコンデザイン |
| アニメーション | 300ms以内のトランジション                              | 300ms以内のトランジション                                  | ストレスフリーな操作感       |

### レスポンシブ仕様

- モバイル (〜768px): 1カラム、コンパクト表示
- タブレット (768px〜1024px): 2カラム、中間レイアウト
- デスクトップ (1024px〜): 3カラム、フル機能表示

### カテゴリカラーシステム

| カテゴリ     | ライトモード       | ダークモード       |
| ------------ | ------------------ | ------------------ |
| 仕事         | blue-100/700/200   | blue-900/400/800   |
| プライベート | green-100/700/200  | green-900/400/800  |

```typescript
// カテゴリ定義
interface Category {
  id: 'work' | 'personal' | 'health' | 'study' | 'other';
  name: string;
  icon: string;
  emoji: string;
  color: {
    light: string;
    dark: string;
  };
}

// タスクデータ
interface Todo {
  id: string; // 一意識別子
  title: string; // タスク名（必須）
  description?: string; // 詳細説明（オプション）
  completed: boolean; // 完了フラグ
  createdAt: Date; // 作成日時
  category: Category['id']; // カテゴリID
}

// 統計データ
interface Stats {
  completionRate: number; // 完了率 (0-100)
  completedCount: number; // 完了済み数
  remainingCount: number; // 残りタスク数
  todayCount: number; // 今日作成されたタスク数
  categoryStats: CategoryStat[]; // カテゴリ別統計
}

interface CategoryStat {
  category: Category;
  total: number;
  completed: number;
  percentage: number;
}
```

## ユーザーフロー

### タスク追加フロー

1. タイトル入力フィールドをクリック
2. 「何に取り組みますか？」にタスク名入力
3. カテゴリをドロップダウンから選択（デフォルト: 仕事）
4. 詳細説明を入力（オプション）
5. Enterキーまたは「タスクを追加」ボタンをクリック
6. slide-inアニメーションでリストの先頭に追加
7. フォームが自動リセット、統計が更新

### タスク完了フロー

1. タスクカードの○アイコンをクリック
2. アイコンがcheck-circle-2に変化（emerald色）
3. タイトルと詳細に取り消し線が追加
4. カードの背景がエメラルド系に変化
5. 「完了」バッジが表示
6. 統計とカテゴリ統計がリアルタイム更新

### タスク削除フロー

1. タスクカードの削除ボタン（ゴミ箱）をクリック
2. 即座にfade-outアニメーションで消える
3. 配列から除外、統計が更新
4. 空状態の場合は専用メッセージを表示

### テーマ切り替えフロー

1. ヘッダーの月/太陽アイコンをクリック
2. 全要素が0.3秒でスムーズに色変化
3. 背景、カード、テキスト色が一斉に変更
4. アイコンが月⇔太陽で切り替わり
5. 設定状態を保持（localStorage推奨）

## 成功指標（KPI）

### 定量指標

- 完了率: 追加されたタスクのうち実際に完了される割合（目標: 70%以上）
- 継続率: 3日連続で使用するユーザー割合（目標: 60%以上）
- 操作時間: タスク1つ追加するまでの時間（目標: 15秒以内）
- カテゴリ利用率: デフォルト以外のカテゴリ使用率（目標: 40%以上）
- 詳細入力率: 詳細説明を入力するタスクの割合（目標: 30%以上）

### 定性指標

- 使いやすさ: "直感的で迷わない"
- 満足度: "毎日使いたくなる美しさ"
- 推奨度: "友人に見せたくなるアプリ"
- 達成感: "完了時に気持ちいい"
